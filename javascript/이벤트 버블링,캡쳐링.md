# 이벤트 버블링 , 켑쳐링

동시에 여러 이벤트가 걸려있을 때 이벤트 순서를 결정하는 방법
html->body>div인 상태에서 div를 클릭하면
아래와 같이 순서대로 캡쳐링이 진행 된 후 버블링으로 올라가면서 이벤트를 감지한다. 

이떄 addEventListenerd에 3번쨰 인자값true or false 에따라 이벤트 순서를 결정할 수 있다.
