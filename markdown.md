## 5.인용문 앞에 >붙이기

ex)


```
> If you can imagine it, you can achieve it; if you can dream it, you can become it.
당신이 상상을 할 수 있다면 그것을 이룰 수 있고, 당신이 꿈꿀 수 있다면 그 꿈대로 될 수 있다.
```
> If you can imagine it, you can achieve it; if you can dream it, you can become it.
당신이 상상을 할 수 있다면 그것을 이룰 수 있고, 당신이 꿈꿀 수 있다면 그 꿈대로 될 수 있다

>> wow >> 두개 씀
>>> wow >>> 세개 씀
>>>> wow $>>> 네개 씀
